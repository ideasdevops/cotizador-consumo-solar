Tipos de Sistemas Solares

Conoce las diferentes opciones de instalación solar

Sistema On-Grid

Conectado a la red eléctrica. Ideal para reducir costos y vender excedentes.

  • Conectado a la red
  • Venta de excedentes
  • Sin baterías
  • Menor costo inicial

Sistema Off-Grid

Independiente de la red. Perfecto para zonas rurales o cortes frecuentes.

  • Independiente de la red
  • Con baterías
  • Mayor autonomía
  • Costo inicial mayor

Sistema Híbrido

Combinación de ambas tecnologías. Máxima eficiencia y respaldo.

  • Conectado + baterías
  • Máximo respaldo
  • Optimización inteligente
  • Costo intermedio
